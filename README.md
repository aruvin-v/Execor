# Execor

Execor is a Windows-based application designed to execute GGUF-based Large Language Models (LLMs) locally, enabling offline inference and efficient model management.

## 🚀 Overview

Execor provides a lightweight and flexible environment to run quantized LLMs without relying on cloud services. It is built to simplify local model execution workflows and improve developer productivity.

## ✨ Features

- Run GGUF-based LLM models locally
- Offline inference support
- Dynamic model execution workflows
- Simple and efficient execution interface
- Designed for internal development and testing use cases

## 🛠 Tech Stack

- .NET (C#)
- WPF (Windows Presentation Foundation)
- GGUF model format
- Local inference runtime

## 📦 Use Cases

- Local AI experimentation
- Offline LLM execution
- Development and testing workflows
- Reducing dependency on external AI services

## ⚙️ How It Works

1. Load a GGUF model into the application
2. Configure execution parameters
3. Run inference locally
4. View results through the application interface
5. Create **models** folder inside project's root folder and place your GGUF models there
